# Game Assets Directory

This folder contains all game assets for Tech Knight.

## Structure

- **characters/** - Character sprites, animations, and turnarounds
- **maps/** - Game environment and level assets
- **ui/** - User interface elements, menus, HUD
- **audio/** - Sound effects, music, and voice lines

## Asset Guidelines

### Character Assets
- Resolution: 2048x2048 for character sheets
- Sprite size: 128x128 pixels per frame
- Format: PNG with alpha transparency
- Animations: 6-frame cycles for walk/run

### Map Assets
- Tileset size: 32x32 pixels
- Format: PNG
- Include collision maps

### UI Assets
- Resolution: @2x for HD displays
- Format: PNG or SVG
- Keep consistent color palette

### Audio Assets
- Format: MP3 or OGG
- Sample rate: 44.1kHz
- Bit rate: 192kbps minimum
