## SPRITE CYCLE - RUNNING ANIMATION (6 FRAMES)

**Side-view running sequence:**

1. **Frame 1**: Contact (Left) - Left foot plants, body leans forward
2. **Frame 2**: Mid-stance (Left) - Weight transfers, right leg swings
3. **Frame 3**: Take-off (Left) - Push-off, both feet momentarily airborne
4. **Frame 4**: Contact (Right) - Right foot plants, momentum carries
5. **Frame 5**: Mid-stance (Right) - Weight on right, left leg swings
6. **Frame 6**: Take-off (Right) - Push-off completes cycle

**Cape physics**: Flows behind character, 1-frame delay for natural motion

## TECHNICAL SPECIFICATIONS

### Art Style
- **Type**: Clean, high-res 2D pixel art / vector hybrid
- **Resolution**: 2048x2048 canvas for character sheet
- **Sprite Size**: 128x128 pixels per frame (game sprites)
- **Layering**: Separate layers for:
  - Base body
  - Armor pieces
  - Weapons
  - Effects (glow, particles)
  - Cape/cloth physics

### Animations Required
1. **Idle**: Breathing animation, subtle armor glow
2. **Walk**: 6-frame cycle (as detailed above)
3. **Run**: 6-frame cycle (faster, more dynamic)
4. **Attack**: 4-frame combo (sword swing)
5. **Block**: Shield raise with glow effect
6. **Special**: Code burst animation
7. **Victory**: Weapon raise, cape flourish
8. **Damage**: Recoil animation

### Special Abilities

#### 1. **Python Strike** (Basic Attack)
- Quick melee combo
- Cooldown: None
- Damage: Standard

#### 2. **AI Assist** (Support Skill)
- Summons AI companion for 10 seconds
- Increases attack speed 20%
- Cooldown: 30 seconds

#### 3. **Firewall Defense** (Shield Skill)
- Blocks all incoming damage for 3 seconds
- Reflects 50% damage to attackers
- Cooldown: 45 seconds

#### 4. **Startup Sprint** (Movement Skill)
- Speed boost 50% for 5 seconds
- Leaves digital trail effect
- Cooldown: 20 seconds

#### 5. **Debug Ultimate** (Ultimate Ability)
- Area-of-effect code burst
- Massive damage + stun
- Cooldown: 120 seconds

### Character Stats (Level 1)
- **Health**: 100 HP
- **Attack**: 15 DMG
- **Defense**: 12 DEF
- **Speed**: 8 SPD
- **Intelligence**: 18 INT (affects AI Assist)
- **Innovation**: 20 INN (unique stat, affects cooldowns)
