## 🎮 GAMEPLAY MECHANICS

### Current Playable Demo Features ✅
- **Movement System**: WASD controls with sprint
- **Combat System**: Space bar to attack enemies
- **Defense System**: Q to activate shield
- **Enemy AI**: Enemies spawn and chase the player
- **Health System**: Take damage, game over at 0 HP
- **Score System**: Earn points for defeating enemies
- **Visual Effects**: Particles, animations, smooth controls

### Controls
- **W/A/S/D** - Move character
- **SHIFT** - Sprint (increased speed)
- **SPACE** - Attack with Code Blade
- **Q** - Toggle Shield (blocks damage)

### Combat System
- **Health**: 100 HP base
- **Armor**: Upgradeable defense system
- **Weapons**: Melee and ranged options
- **Abilities**: Cooldown-based special moves

### Match Flow
1. **Lobby**: Team formation & character selection
2. **Drop**: Spawn into the map
3. **Loot**: Collect weapons, armor, power-ups
4. **Combat**: Engage enemies, use abilities
5. **Victory**: Last team standing wins

### Progression System
- **Level Up**: Gain XP from matches
- **Unlock Rewards**: Skins, emotes, badges
- **Leaderboard**: Track rankings
- **Achievements**: Complete challenges
